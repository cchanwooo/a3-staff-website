# A3 Staff Solutions Website

This is a modern business website for **A3 Staff Solutions LLC**, a staffing agency specializing in skilled trades.

## Tech Stack
- **React**: Frontend library
- **Vite**: Build tool
- **TailwindCSS**: Utility-first CSS framework
- **Framer Motion**: Animations
- **React Router**: Navigation
- **React Icons**: Iconography

## Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)

### Installation

1.  Clone the repository or navigate to the project directory:
    ```bash
    cd a3-staff-website
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

## Project Structure
- `src/components`: Reusable UI components (Navbar, Footer, Button, Card, Input)
- `src/pages`: Individual page components (Home, Services, About, Apply, ClientRequest, Contact)
- `src/layouts`: Layout wrapper components
- `src/assets`: Static assets

## Customization
- **Colors**: Brand colors are defined in `tailwind.config.js`.
- **Content**: Update text and images in the respective page components.
